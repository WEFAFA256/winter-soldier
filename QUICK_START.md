# 🚀 Quick Start - DPO Pay Integration

## ⚡ TL;DR

**PesaPal → DPO Pay migration complete!**

---

## 🎯 What You Need

### 1. DPO Pay Account
Sign up: https://dpogroup.com

### 2. Get Credentials
From DPO dashboard:
- Company Token
- Service Type ID

### 3. Add to .env
```bash
DPO_COMPANY_TOKEN=your_token_here
DPO_SERVICE_TYPE=your_service_type_here
```

---

## 🏃 Run It

```bash
# Terminal 1 - Frontend
npm run dev

# Terminal 2 - Backend  
npm run server:dpo
```
